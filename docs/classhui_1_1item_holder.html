Detailed Description

template<typename T>
class hui::itemHolder< T >

A slot which accepts an item as a child.

Note
ItemHolder template type must match the item's template type for the item to be held
See also
hui::itemHolder::setAcceptFunction

template<typename T >
void hui::itemHolder< T >::setAcceptFunction ( bool(*)(itemHolder< T > *, item< T > *)  func)
inline

Sets the function to check if an item is valid for this holder.

If the function returns true and the holder has no current child, the item will be accepted

Parameters
funcFunction which is given
Note
This function will also be called to check if a child can be removed.
Warning
While the itemHolder<T>* paramter will never be a nullptr, the item<T>* parameter will be nullptr when checking if a child can be removed.
