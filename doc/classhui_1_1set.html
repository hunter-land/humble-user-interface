Detailed Description

A collection visible elements.

This is the container through which elements are managed and interacted with.

Using the functions render(), userLogic(), resetUserLogic(), loopLogic(), and resetLoopLogic() will cause the set to loop through all contained GUI elements, causing the same function to be called for each element.

Note
Render order is from firstly added element to lastly added element. Logic order is the opposite of render order.
Sets can be nested, providing sub-windows such as a scrolling menu
All elements must be inside of a set to work correctly.
See also
hui::scrollBar
